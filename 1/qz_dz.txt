как работаем с ссылками в flask    в скрипте импортируем url_for, в шаблоне пользуем элемемент jinja2 а не хардовую ссылку
from flask import url_for => <a href = "{{ url_for('function_name' }}"

что ложится в шаблоне jinja в url_for    имя функции из py-скрипта

как добавить css-файл    положить его по адресу статичных файлов фласк или указать где эта папка через арг-т Flask
app=Flask(__name__,static="path") => <link rel="stylesheet" href= "{{ url_for('static', filename='main.css') }}">

родительский шаблон    это обычная веб-страница, которая в местах с уникальным контентом содержит блоки

блок шаблона    {% block block_name %} content {% endblock %}

требование к именам блоков    уникальное на уровне одного шаблона, одинаковые имена для одного блока между разными шаблонами
